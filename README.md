UVCAndroid
=========

Library and sample to access UVC camera on non-rooted Android device

How do I use it?
---

### Setup

##### Dependencies
```groovy
repositories {
  mavenCentral()
}

dependencies {
    implementation 'com.herohan:UVCAndroid:1.0.1'
}
```
Requirements
--------------
Android 5.0+
