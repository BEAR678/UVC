UVCAndroid
=========

Library and sample to access UVC camera on non-rooted Android device

### Setup

##### Dependencies
```groovy
repositories {
  mavenCentral()
}

dependencies {
    implementation 'com.herohan:UVCAndroid:1.0.2'
}
```
